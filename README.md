# Install
1. Get [Glamorous Toolkit](https://gtoolkit.com/).

2. Open a playground and evaluate:
```Smalltalk
Metacello new 
  baseline: 'Metamath'; 
  repository: 'github://Peter-H-Meadows/GtMetamath';
  load.
```
2
![gif](images/install.png)


```Smalltalk
  Metamath open .
```
